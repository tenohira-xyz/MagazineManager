# MagazineManager
雑誌の目次を管理するためのwebシステム

## メモ
- 雑誌一覧画面から雑誌詳細、目次、索引へ遷移する
- 雑誌詳細ページに設定項目を表示するか設定ページへのリンクを表示する
- 雑誌詳細ページからデータをエクスポートできるようにする。
